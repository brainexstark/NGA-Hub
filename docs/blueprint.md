# **App Name**: NGA Hub

## Core Features:

- User Segmentation: Categorize users into three age-specific groups: under 10, 10-16, and 16+.
- Age-Specific Content Feeds: Curate separate content feeds for each user segment, using AI to filter and recommend content.
- AI-Powered Content Moderation: Utilize Grok-4, Sycamore, and GPT to moderate content and ensure it aligns with the appropriate age group's guidelines. This feature uses these AIs as tools.
- Learning Integration: Integrate educational content suitable for each age group into the platform.
- Entertainment Integration: Integrate entertaining content suitable for each age group into the platform.
- Social Interaction Features: Implement standard social media features like followers, following, and disciples (customizable based on age group).
- Personalized AI Assistant: Implement an assistant capable of offering feedback about content.
- Video Bank: Store videos instead of downloads.
- Withdraw Video: Allow users to withdraw videos from their account.
- Deposit Video: Allow users to deposit videos into their account.
- Channel Creation: Page for creating channels.
- Instagram Features: Include all features possessed by Instagram.
- Direct Website Access: Has direct access to all other websites, social media platforms, and free games based on age.
- Age-Based Operation: The app works based on age bracket as specified by the user at sign-in.
- Premium Features: Offers premium features available through in-app purchases.
- AI Live Streaming: Page for AI live streaming of lessons.
- Access Open AIs: Page for accessing open AIs.
- Feature Request: Page for users to comment on features they want to see next.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust, communication, and technology.
- Background color: A light gray (#F0F0F0), promoting neutrality and readability.
- Accent color: A complementary orange (#FF8C00) to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk', a sans-serif font lending a tech-centric appearance.
- Body font: 'Inter', sans-serif to maintain consistency in readability and modernity.
- Use Sci-Fi themed icons for the app. The icons should be suitable for all age groups; design varies slightly per age group.
- Employ subtle animations for content transitions and user interactions to enhance engagement.
- Stylish UI for all HTML pages.
